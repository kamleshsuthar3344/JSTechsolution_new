# Roadmap: Affiliate and Referral Program Backend

This document outlines the business model and the technical roadmap for implementing the backend for Earning Sharthi's Affiliate and Referral programs.

## 1. Business Model Analysis

### Affiliate Program
*   **Target:** Influencers, Bloggers, and Content Creators.
*   **Goal:** Promote Earning Sharthi services (Web Dev, SEO, Marketing).
*   **Commission:** Percentage-based (e.g., 15-20%) on successful sales.
*   **Tracking:** Unique affiliate links with cookie-based tracking.

### Referral Program
*   **Target:** Existing clients and individual partners.
*   **Goal:** Refer new clients or partners to the platform.
*   **Commission:** Fixed amount or percentage (e.g., 25%) per successful referral.
*   **Tracking:** Referral codes or direct links.

---

## 2. Technical Roadmap

### Phase 1: Database Design (MySQL)
The foundation of the programs will be a robust database schema.

*   **`users` table:** Store user details, including their unique `referral_code` and `affiliate_id`.
*   **`referrals` table:** Track who referred whom and the status of the referral.
*   **`conversions` table:** Record successful sales/leads generated by affiliates.
*   **`payouts` table:** Track payment history and pending requests for partners.

### Phase 2: Backend Development (PHP)
Simple and efficient PHP scripts to handle the logic.

*   **Auth System:** Registration and Login for partners.
*   **Tracking Engine:** 
    *   Handle affiliate link clicks (set cookies).
    *   Validate referral codes during signup.
*   **Dashboard Logic:**
    *   Calculate earnings and total referrals.
    *   Display unique links and marketing materials.
*   **Payout Request System:** Allow partners to request payouts once they reach a threshold.

### Phase 3: Simple UI (HTML/CSS)
Lightweight, SEO-friendly pages for the partner portal.

*   **Landing Pages:** High-converting pages explaining both programs.
*   **Partner Dashboard:** Simple interface to see stats, links, and earnings.
*   **Admin Panel:** (For Internal Use) To manage partners, approve payouts, and view reports.

### Phase 4: Integration
Linking the new PHP system with the existing React website.
*   Update "Partnership" dropdown links in React to point to the new PHP subfolder.
*   Ensure shared branding (CSS/Logo) between React and PHP pages.
